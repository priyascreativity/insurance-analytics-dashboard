# Insurance Policy Data Simulation & Analytics

## Project Overview
This project simulates car insurance policy sales and claims data using Python. The dataset contains 1,000,000 simulated policy records.

## Tools Used
Python (Pandas, NumPy)
Power BI
Data Visualization

## Key Analysis
- Total premium collected
- Total claim cost
- Monthly claim trends
- Claim cost by policy tenure
- Premium collected by tenure
